Technologies et langages
========================

Langage
-------

- **Python 3.11**

Framework
---------

- **Django 3.0** — framework web principal

Base de données
---------------

- **SQLite3** — base de données relationnelle légère (développement et production)

Serveur de production
---------------------

- **Gunicorn** — serveur WSGI pour la production
- **WhiteNoise** — service des fichiers statiques en production

Conteneurisation
----------------

- **Docker** — conteneurisation de l'application
- **Docker Hub** — registre d'images Docker

CI/CD
-----

- **GitHub Actions** — pipeline d'intégration et de déploiement continus

Hébergement
-----------

- **Render** — plateforme de déploiement cloud

Surveillance
------------

- **Sentry** — surveillance des erreurs et journalisation

Gestion des variables d'environnement
---------------------------------------

- **python-decouple** — séparation de la configuration du code

Documentation
-------------

- **Sphinx** — génération de la documentation technique
- **Read the Docs** — hébergement de la documentation
- **sphinx-rtd-theme** — thème Read the Docs pour Sphinx

Outils de développement
------------------------

- **pytest** / **pytest-django** — tests unitaires et d'intégration
- **pytest-cov** — mesure de la couverture de test
- **flake8** — linting et vérification du style de code
